.gerdbakke File Program

This project is completly opensource. It can create encrypted .gerdbakke files. .gerdbakke files are similar to .txt files, but encrypted. Thanks for downloading!

IMPORTANT! You probably must run this command in cmd (to open: windows key + r, then type cmd, then hit enter): pip install cryptography
